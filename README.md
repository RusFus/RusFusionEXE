# RusFusionEXE

Русификатор интерфейса Autodesk Fusion 360 для Windows

**Примечание:** Это неофициальный проект. Автор программы не связан с компанией Autodesk.

## Как установить

1. Скачайте архив с программой
2. Распакуйте в отдельную папку
3. Запустите файл `RusFus.exe`

## Требования

- Windows 10/11
- Autodesk Fusion 360
- Интернет-соединение (для проверки обновлений)

## Автор

RusFus
